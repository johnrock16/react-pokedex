# react-pokedex

React pokedex is a simple Web Apllication which use PokeAPI to fill all screen with pokemons and a short text about their.

## Installation

Use npm to install react-pokedex.

```bash
npm install
```

## Usage

Client:

``` bash
npm run client
```

Server:
``` bash
npm run server
```

Both:
``` bash
npm run dev
```
